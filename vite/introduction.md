# Vite

Vite is a frontend build tool that provides a fast and lean development experience for modern web projects. It contains two main components:
  1. A development server. Converts alien syntax like react and typescript to javascript that the browser will understand and serves that. Checks for file changes and reloads changed files instantly.
  2. A build command that bundles your code, pre-configured to output highly optimized static assets for production.

## Basic Workflow

```
# Create project
npm create vite@latest

# Install dependencies
npm install

# Start dev server
npm run dev

# Build production app
npm run build

# Preview production build
npm run preview
```
